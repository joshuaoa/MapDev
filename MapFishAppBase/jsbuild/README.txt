To build the JavaScript code in a single, compressed file run the
following commands in the current directory:

$ jsbuild -o ../mapfishappbase/public/build/mapfish app.cfg
$ cp -r ../mapfishappbase/public/mfbase/mapfish/img ../mapfishappbase/public/build/mapfish/
$ cp -r ../mapfishappbase/public/mfbase/openlayers/img ../mapfishappbase/public/build/openlayers/
$ cp -r ../mapfishappbase/public/mfbase/openlayers/theme ../mapfishappbase/public/build/openlayers/
